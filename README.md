## Hva er Nabodo?

Nabodo er en åpen kildekode-plattform der folk kan registrere seg som verter og tilby toalettet sitt til naboer i nærheten. Gratis. Ingen app. Bare en dør som åpnes.

Oslo har systematisk for få offentlige toaletter i boligstrøk. Nabodo løser dette nedenfra — én dør om gangen.

```bash

```

Ingen avhengigheter. Bare åpne `index.html` i nettleseren.
