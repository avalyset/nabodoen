# Bidra til Nabodo

Takk for at du vil bidra! Nabodo er et åpent civic tech-prosjekt og vi ønsker alle typer bidrag velkommen.

## Kom i gang

```bash
git clone https://github.com/nabodo/nabodoen.git
cd nabodoen
open index.html
```

Ingen avhengigheter. Bare åpne `index.html` i nettleseren.

## Slik bidrar du

1. **Fork** repo-en på GitHub
2. **Lag en branch** med beskrivende navn:
   - `fix/beskrivelse` for bugfixes
   - `feature/beskrivelse` for nye funksjoner
   - `docs/beskrivelse` for dokumentasjon
3. **Gjør endringene dine**
4. **Test** at `index.html` åpnes uten feil i nettleseren
5. **Commit** med tydelig melding: `git commit -m "fix: rett skrivefeil i footer"`
6. **Push** og åpne en **Pull Request** mot `main`

## Commit-meldinger

Vi bruker [Conventional Commits](https://www.conventionalcommits.org/):

| Prefix | Bruk |
|--------|------|
| `feat:` | Ny funksjonalitet |
| `fix:` | Bugfix |
| `docs:` | Dokumentasjon |
| `style:` | CSS/design, ingen logikkendring |
| `refactor:` | Kodestruktur uten ny funksjonalitet |
| `chore:` | Vedlikehold, dependencies |

## Hva vi trenger hjelp til

Se åpne Issues med label `good first issue` for gode startpunkter.

Prioriterte områder:
- 🗺️ Kartverket API-integrasjon
- 🔐 BankID via Criipto
- 📱 Mobiltilpasning og tilgjengelighet
- 🌍 Engelsk oversettelse
- 🧪 Testing

## Kodestil

- Ren HTML/CSS/JS — ingen rammeverk i fase 1
- Norsk som hovedspråk i UI-tekster
- Kommentarer på norsk eller engelsk
- Universell utforming (WCAG 2.1 AA)

## Spørsmål?

Åpne en Issue med label `question` eller bidra i diskusjoner på GitHub.
